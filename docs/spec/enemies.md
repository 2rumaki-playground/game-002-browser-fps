# 敵タイプ仕様

## 基本敵（Chaser）

現在実装されている唯一の敵タイプ。

### パラメータ

[constants.md](./constants.md) 参照。

### 見た目

- 赤色の球体（diameter: [constants.md](./constants.md)）
- 自発光あり（emissive）

### 行動パターン

| 条件                     | 行動                          |
| ------------------------ | ----------------------------- |
| プレイヤーとの距離 > 近接範囲 | プレイヤーに向かって直進      |
| プレイヤーとの距離 ≤ 近接範囲 | 近接攻撃（DPSダメージ）       |

### 移動AI

- 毎フレーム、プレイヤー位置との差分ベクトルを計算
- Y成分を 0 にして水平方向のみ移動
- 正規化して移動速度を掛ける
- 障害物回避なし（壁をすり抜ける場合がある）

### スポーン

- 現在は `src/fps/createGame.ts` 内で固定位置に3体スポーン
- 倒された敵のリスポーンは未実装
