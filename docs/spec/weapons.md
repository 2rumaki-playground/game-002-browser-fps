# 武器仕様

## 基本射撃（現在の唯一の武器）

| 項目           | 値                                     |
| -------------- | -------------------------------------- |
| 発射方式       | 単発（左クリック）                     |
| 判定方式       | Raycast（カメラ前方）                  |
| ダメージ       | [constants.md](./constants.md) 参照    |
| 射程           | [constants.md](./constants.md) 参照    |
| クールダウン   | [constants.md](./constants.md) 参照    |
| ヒット判定対象 | `mesh.metadata?.isShootable === true` のメッシュ |

### ヒット時の挙動

1. 対象の HP からダメージを減算
2. HUD のクロスヘアが拡大（ヒットフィードバック）
3. HP が 0 以下になった場合、メッシュを破棄しキルカウンターを加算

### 今後の拡張候補

- 複数武器の切り替え
- 弾数制限・リロード
- 武器ごとの射撃レート・ダメージ差
- 弾道表現（トレーサー）
