# マップ仕様

## 現在のマップ：アリーナ

閉鎖的な正方形の部屋。四方を壁で囲まれ、内部に柱が配置されている。

### 構成要素

| 要素   | 説明                                                    |
| ------ | ------------------------------------------------------- |
| 地面   | 平面メッシュ。衝突判定あり                              |
| 壁     | 4面のボックスメッシュ。衝突判定あり                     |
| 柱     | 円柱メッシュ。遮蔽物として機能。衝突判定あり            |

### サイズ

[constants.md](./constants.md) 参照。

### 柱の配置

6本の柱が対称的に配置されている。プレイヤーが遮蔽物として利用可能。

配置パターン（[constants.md](./constants.md) 参照）：
- 四隅付近に4本
- 中央軸上に2本

### ライティング

- 半球光源（HemisphericLight）1灯
- 上方向から照射
