# 定数一覧

**本ファイルがすべての数値パラメータの Single Source of Truth である。**

他の仕様書から数値を参照する場合は、必ず本ファイルへリンクすること。

## プレイヤー

| 定数名           | 値     | 説明                     |
| ---------------- | ------ | ------------------------ |
| PLAYER_HP        | 100    | プレイヤー初期HP         |
| PLAYER_HEIGHT    | 1.8    | カメラ高さ（Y座標）      |
| PLAYER_SPEED     | 0.65   | 移動速度                 |
| PLAYER_ELLIPSOID | 0.4, 0.9, 0.4 | 衝突判定楕円体     |
| ANGULAR_SENSIBILITY | 3200 | マウス感度（値が大きいほど遅い） |

## カメラ

| -------- | ---- | ------------------ |
| MIN_Z    | 0.1  | ニアクリップ面     |
| MAX_Z    | 200  | ファークリップ面   |
