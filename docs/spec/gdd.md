# ゲームデザインドキュメント

## 体験目標

| 項目                 | 目標値                                                  |
| -------------------- | ------------------------------------------------------- |
| 1プレイの所要時間     | 3〜5分                                                  |
| 操作習得までの時間   | 30秒以内                                                |
| ターゲットFPS        | 60fps                                                   |

## 設計方針

### アクション性 vs 戦略性

**アクション性を優先する。** FPSとしてのエイム・移動の手応えを最優先とし、戦略要素は最小限に留める。

### シンプルさ vs 深さ

**シンプルさを優先する。** PoC段階では複雑な武器切り替えやスキルツリーは導入せず、基本操作のみで楽しめることを検証する。

### 公平性 vs 爽快感

**爽快感を優先する。** 敵はプレイヤーより弱く設定し、倒す快感を重視する。

## 操作体系

| 入力              | アクション     |
| ----------------- | -------------- |
| W / A / S / D     | 前後左右移動   |
| マウス移動        | 視点操作       |
| 左クリック        | 射撃           |
| ESC               | ポインターロック解除 |
