# コアゲームルール

## ゲーム開始

1. ブラウザでページを開く
2. 画面クリックでポインターロックを取得
3. プレイヤーは初期位置（[constants.md](./constants.md)）にスポーン
4. 敵が固定位置にスポーン

## プレイヤー

- 一人称視点のFPSカメラ
- WASD移動、マウスで視点操作
- 壁・柱・地面との衝突判定あり
- 重力が適用される
- HP を持ち、敵の近接攻撃でダメージを受ける
- HP が 0 以下になると画面が赤くなる（現在は死亡処理未実装）

## 射撃

- 左クリックで射撃
- カメラ前方にRaycastを飛ばし、`isShootable` フラグを持つメッシュにヒット判定
- 射程距離あり（[constants.md](./constants.md)）
- クールダウンあり（[constants.md](./constants.md)）
- ヒット時にクロスヘアが拡大するフィードバック
- ダメージ量は固定（[constants.md](./constants.md)）

## 敵

- 球体メッシュで表現
- プレイヤーに向かって移動する追跡AI
- 近接攻撃：一定距離内に入るとプレイヤーにDPS（毎秒ダメージ）を与える
- HP を持ち、射撃でダメージを受ける
- HP が 0 以下で消滅し、キルカウンターが増加

詳細は [enemies.md](./enemies.md) を参照。

## HUD

- **クロスヘア**：画面中央に表示。ヒット時に拡大アニメーション
- **HP表示**：画面左上にプレイヤーの現在HP
- **キルカウンター**：画面右上に撃破数
- **ダメージフラッシュ**：被ダメージ時に画面が赤く明滅

## 勝敗条件

現在のPoC段階では勝敗条件は未定義。敵を全滅させてもゲームは継続する。

## ポインターロック

- 画面クリックでポインターロックを取得
- ロック中のみ操作を受け付ける
- ESCキーでロック解除
- ロック解除中はオーバーレイを表示
